# pardus-build

Pardus 21 Backports iso yapan scprit

Önce bash scpritlerin program olarak çalışmasına izin verelim

chmod +x xfce.sh gnome.sh kde.sh cinnamon.sh mate.sh lxde.sh

Aşağıdaki komutu uçbirime yazmanız yeterli :)

Xfce için:
sudo ./xfce.sh 

Gnomeiçin:
sudo ./gnome.sh

Kde için: 
sudo ./kde.sh

Cinnomon için:
sudo ./cinnamon.sh

Mate için:
sudo ./mate.sh

Lxde için:
sudo ./lxde.sh
